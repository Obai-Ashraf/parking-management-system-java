# Parking Lot Management System

## 🚗 Overview
A Java-based Parking Lot Management System with a graphical user interface (GUI) built using Java Swing. The system manages vehicle entry, exit, and parking slot allocation.

---

## ⚙️ Features
- Vehicle entry and exit tracking
- Parking slot management
- Simple and interactive GUI
- Basic data handling for vehicles

---

## 🧠 Concepts Used
- Object-Oriented Programming (OOP)
- Classes and Objects
- Encapsulation
- GUI development with Java Swing

---

## 🛠️ Technologies Used
- Java
- Java Swing
- OOP principles
